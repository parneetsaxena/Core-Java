# Core Java

This repository contains my Core Java practice focused on building strong fundamentals through hands-on coding.

It includes implementations of key Java concepts, problem-solving exercises, and small programs organized topic-wise.

## What This Repository Covers

- Arrays and basic problem solving
- Object-Oriented Programming (OOP)
- Collections Framework (ArrayList, HashMap, etc.)
- Exception Handling
- File Handling
- Core Java utilities and logic building Structure

The repository is organized based on topics and practice areas.

## 📂 Folder Structure

- "advanced/" → File Handling, Multithreading, Serialization, Annotations
- "basics/" → Syntax, Variables, Problem Solving
- "collections/" → List, Map, Set, Queue, Generics, Comparator, Comparable
- "loops/" → Pattern problems (Star Patterns, etc.)
- "OOPs/" → Classes, Objects, Inheritance, Polymorphism, Abstraction, Encapsulation



*Core-Java/ │ ├── advanced/ ├── basics/ ├── collections/├── loops/ ├── OOPs/*

(Structure will evolve as more topics are added)

## How to Run

- Navigate to the folder containing the Java file
- Compile the file: javac FileName.java
- Run the program: java FileName
